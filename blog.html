Answering the Questions

  1. Difference between map(), filter(), forEach() and find() .

    The map() function executes the same code on every element in an array and returns a new array with the updated elements.

    Every element in an array is examined by filter() to see if it satisfies a certain set of requirements before returning a new array that only contains the elements that do.

    The forEach() method calls a function once for each element in an array, in order.

    The find() method does not alter the original array; it returns the value of the first element that passes a test, executes a function for each element in the array, returns undefined if no elements are discovered, and does not execute the function for empty elements.

  2. Difference between localStorage , Session storage and Cookies .

    By the localStorage method we can save and retrieve data in the browser using a sort of web storage object that is provided . Data can be kept and accessed without expiring; it will be accessible even after a visitor leaves your website.

    sessionStorage is similar to localStorage; the difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends.

    Cookies are text files that include tiny bits of information, such as a login and password, and are used to recognize your machine when you're connected to a computer network. HTTP cookies are a particular type of cookie that are used to identify specific users and enhance web browsing. The server creates data in a cookie as soon as you connect. An ID that is specific to you and your computer is used to identify this data.

  3. What is event loop ? How does it works ?

    Event loop, which is in charge of running the program, gathering and analyzing events, and carrying out queued sub-tasks.

    The Call Stack and the Callback Queue are two simple tasks performed by the Event Loop. The Event Loop will pull the first event from the queue and put it to the Call Stack if it is empty, where it is effectively run.
